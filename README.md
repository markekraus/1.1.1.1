# 1.1.1.1

A PowerShell Module for with 1.1.1.1's DNS over HTTPS (DoH) Web Service.

## Quick Start

```powershell
Install-Module 1.1.1.1 -Scope CurrentUser
Import-Module 1.1.1.1
Resolve-1111 google.com
```

## Installation

1.1.1.1 is available on the PowerShell Gallery.

```powershell
Install-Module 1.1.1.1 -Scope CurrentUser
```
